# opal-router

A simple router for Opal applications.

## Installation

Add to your Gemfile:

```ruby
gem 'opal-router', :git => 'git://github.com/adambeynon/opal-router.git'
```

Then, anywhere in your opal code:

```ruby
require "opal-router"
```
